# Paresh Enterprises Backend - Simple Version

## Quick Start

1. Extract backend folder
2. Run: `docker-compose up -d`
3. Visit: http://localhost:8000/docs

## Features
- ✅ No Pydantic schema errors
- ✅ Simple, working authentication
- ✅ MongoDB with Motor
- ✅ FastAPI with CORS
- ✅ JWT tokens
- ✅ Ready for React

## Endpoints
- GET /health
- GET /docs
- POST /auth/register
- POST /auth/login  
- GET /auth/profile
- GET /auth/me
- POST /auth/logout
